About my code

A longstanding goal of mine is to write a computer program KhCob that calculates the induced maps on Khovanov homology for a given link cobordism. As part of my dissertation, I wrote a preliminary version of the KhNoDe program (below) for determining nontriviality of Khovanov homology classes. My current work focuses on adapting this program to write a preliminary version of KhCob.
I usually code in Java. Recently I learned Python and SageMath to better support the math behind the programs below. I hope to someday learn C++.

About my code

A longstanding goal of mine is to write a computer program KhCob that calculates the induced maps on Khovanov homology for a given link cobordism. As part of my dissertation, I wrote a preliminary version, called KhNoDe (see below), which determines the nontriviality of Khovanov homology classes. I am currently updating this program to a range of applications, including finding nontrivial Khovanov homology classes.

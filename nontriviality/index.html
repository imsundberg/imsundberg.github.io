Nontriviality of Khovanov homology classes

This SageMath program allows you to determine nontriviality of Khovanov homology classes by determining if the chain element representing the class is a boundary in the Khovanov chain complex. My recent research has used this program to distinguish the Khovanov-Jacobsson classes associated to pairs of slice disks bounding the knots 6_1 and 9_46. Many additional applications exist, so I want to share this program with the topology community - enjoy!
A paper will soon be uploaded to the arXiv which details the implementation of this program.

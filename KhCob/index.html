KhCob

Khovanov homology Cobordism induced map calculator
This Python program is currently being written. It will allow the user to calculate the cobordism induced map on Khovanov homology for a given surface. More precisely, it will take as input (1) a link cobordism S represented as a movie and (2) a cycle c from the Khovanov chain complex; it will output the cycle Kh(S)(c).
